# Offline File Explorer Generator 🗂️

เครื่องมือสำหรับสแกนโครงสร้างโฟลเดอร์และไฟล์ (File/Folder Structure) ของระบบใดๆ แล้วสร้างออกมาเป็นเว็บไซต์หน้าเดียว (Single-Page HTML) ที่สามารถนำไปเปิดดูบนคอมพิวเตอร์เครื่องอื่นผ่าน Web Browser (เช่น Chrome, Edge, Safari) ได้ทันที โดยจำลองหน้าตาและการทำงานคล้ายคลึงกับ File Explorer ของจริง

มีประโยชน์มากในกรณี:
- ต้องการเก็บประวัติหรือสำรวจโครงสร้างของระบบ, โฟลเดอร์สำรองข้อมูล (Backup)
- ต้องการส่งภาพรวมของไฟล์ทั้งหมดไปให้เพื่อนร่วมงานหรือทีมดู โดยไม่ต้องอัปโหลดไฟล์จริงทั้งหมดที่มีขนาดใหญ่
- ระบบต้นทางอยู่ใน Server ตีบตัน ที่ไม่สามารถ Copy ไฟล์ออกมาได้ทั้งหมด แต่รันสคริปต์สแกนแค่ชื่อไฟล์ออกมาได้

## 🛠️ ไฟล์ในโปรเจกต์นี้
*   **`Export-Structure.ps1`**: สคริปต์ PowerShell ที่ใช้สำหรับสแกนโฟลเดอร์ และสร้างผลลัพธ์
*   **`explorer_template.html`**: ไฟล์เทมเพลตต้นฉบับ ที่ใช้จัดหน้าตาเว็บ (UI) และการทำงาน (มีระบบรองรับการ Drag & Drop)
*   *`.gitignore`*: ตั้งค่าไว้เพื่อไม่ให้ไฟล์ผลลัพธ์ (`OfflineExplorer.html` และ `structure.json`) ถูก Push ขึ้นไปเกะกะบน Git

## 🚀 วิธีใช้งาน

### 1. วิธีสร้างหน้าเว็บ Explorer (รันสคริปต์บนเครื่องที่มีไฟล์จริง)
1. นำไฟล์ `Export-Structure.ps1` และ `explorer_template.html` ไปวางไว้ในเครื่องหรือโฟลเดอร์เป้าหมาย
2. คลิกขวาที่ไฟล์ **`Export-Structure.ps1`** แล้วเลือก **"Run with PowerShell"**
3. หน้าต่าง PowerShell จะเด้งขึ้นมาถามว่าต้องการสแกน Path ไหน:
    *   *กด `Enter` เลย* ถ้าต้องการสแกนโฟลเดอร์ปัจจุบันที่ไฟล์นี้วางอยู่
    *   *พิมพ์ Path เต็มๆ* (เช่น `D:\MyBackup\DB`) เพื่อสแกนโฟลเดอร์อื่น แล้วกด `Enter`
4. รอสคริปต์ทำงานจนเสร็จ เมื่อเสร็จสิ้น คุณจะได้ไฟล์ใหม่โผล่ขึ้นมา 2 ไฟล์:
    *   **`structure.json`**: ไฟล์ข้อมูลดิบที่เก็บข้อมูลโครงสร้างไฟล์และโฟลเดอร์
    *   **`OfflineExplorer.html`**: ไฟล์เว็บไซต์แบบสมบูรณ์ ที่ฝังข้อมูล JSON ชุดนี้เข้าไปเรียบร้อยแล้ว

### 2. วิธีใช้งานฝั่งผู้รับ (คนที่จะเปิดดู)
เพียงแค่ส่งไฟล์ **`OfflineExplorer.html`** เพียงไฟล์เดียว ไปให้ผู้รับ ผู้รับสามารถ:
1. **ดับเบิลคลิกเปิดไฟล์ `OfflineExplorer.html`** ด้วย Web Browser ใดก็ได้
2. สามารถกดสำรวจโฟลเดอร์, ดูขนาดไฟล์, วันที่สร้าง ได้เหมือน File Explorer ปกติ
3. **การสลับมุมมอง (View Mode):** ที่มุมขวาบน จะมี Dropdown ให้เลือกสลับ 2 โหมด:
    *   **🗂️ Explorer View:** (ค่าเริ่มต้น) มุมมองแบบคลิกเข้าไปดูทีละชั้น พร้อมแถบ Address bar ย้อนกลับ (`Up`) หรือกลับหน้าแรก (`Home`)
    *   **🌳 Tree View:** มุมมองกิ่งไม้ แสดงโฟลเดอร์แบบรวบยอด กดลูกศร `▶` เพื่อกางดูไส้ในแต่ละชั้นได้เรื่อยๆ

> **💡 ทริคเพิ่มเติม (Drag & Drop):**
> หากคุณมีเฉพาะไฟล์ `structure.json` หรือมีการอัปเดตไฟล์ JSON มาใหม่ 
> คุณสามารถเปิดหน้าตาเปล่าๆ ของไฟล์ `explorer_template.html` ขึ้นมาบน Browser แล้ว **ลากไฟล์ `structure.json` ไปทับบนหน้าเว็บ (Drag & Drop)** หน้าเว็บจะทำการอัปเดตข้อมูลขึ้นมาโชว์ให้ทันทีโดยไม่ต้องรันสคริปต์ซ้ำ!
